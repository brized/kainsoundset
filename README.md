# kainsoundset
A soundset for Baldur's Gate, Icewind Dale, and other Infinity Engine games featuring Kain from the Legacy of Kain series.
It fits an evil character, especially a bard.

---

Installation
Extract the contents to your install folder, then run setup-kain_soundset.exe.

The install can be at any point in your mod install order, though I put it at the end. There aren't any known incompatibilities.
